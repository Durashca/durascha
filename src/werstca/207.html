Главная и поперечная оси в флекс блоках в CSS

№1 Пусть главная ось горизонтальна. Куда будет направлена поперечная ось? Ответ: она направленна с верху вниз

№2 Пусть главная ось вертикальна. Куда будет направлена поперечная ось? Ответ: она направлена слева направо

№3 Пусть поперечная ось направлена направо. Куда в таком случае может быть направлена главная ось? Ответ: она направленна с верху вниз

№4 Пусть поперечная ось направлена вниз. Куда в таком случае может быть направлена главная ось? Ответ: она направленна слева направо

№5 Может ли поперечная ось быть направлена справа налево? Ответ: нет, конечно

№6 Может ли поперечная ось быть направлена снизу вверх? Ответ: да

№7 Пусть свойство flex-direction имеет значение row. Куда будет направлена главная ось? Куда будет направлена поперечная ось? Ответ: row( главная направаленна будет слева направо, а поперечная сверху вниз)

№8 Пусть свойство flex-direction имеет значение column. Куда будет направлена главная ось? Куда будет направлена поперечная ось? Ответ: column( главная направаленна будет сверху вниз, а поперечная слева направо)

№9 Пусть свойство flex-direction имеет значение row-reverse. Куда будет направлена главная ось? Куда будет направлена поперечная ось? Ответ: row-reverse(главная будет направленна справо налево, а поперечная сверху вниз)

№10 Пусть свойство flex-direction имеет значение column-reverse. Куда будет направлена главная ось? Куда будет направлена поперечная ось? Ответ: row-reverse(главная будет направленна снизу вверх, а поперечная слево направо)
